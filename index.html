    Tentang Saya

    Saya adalah seorang pelajar yang tertarik di dunia pemrograman dan pengembangan game. Saat ini, saya aktif belajar membuat website, mendesain antarmuka, dan mengembangkan game sederhana secara mandiri.

    Saya terbiasa menggunakan HTML, CSS, JavaScript, dan beberapa tools lain seperti Godot dan Canva. Bagi saya, membuat sesuatu dari nol dan melihatnya berjalan dengan baik adalah hal yang menyenangkan dan memotivasi.

    Saya percaya bahwa kemampuan bisa berkembang lewat konsistensi dan rasa ingin tahu. Karena itu, saya terus belajar dan mencoba hal-hal baru di bidang teknologi.

    DevConnect – Platform Terintegrasi Android & Web
    DevConnect adalah proyek kolaboratif yang menggabungkan pengembangan aplikasi mobile (Android) dan web dalam satu ekosistem. Tujuan utama dari proyek ini adalah menciptakan aplikasi yang responsif, interaktif, dan dapat berjalan di berbagai perangkat dengan pengalaman pengguna yang optimal.
    Fitur Web
    Frontend dibangun dengan HTML, CSS, dan JavaScript untuk antarmuka yang modern dan dinamis. Backend menggunakan PHP dan MySQL untuk mengelola data pengguna, autentikasi, dan sistem notifikasi. Sistem manajemen akun dan dashboard interaktif yang terhubung dengan versi mobile.
    Fitur Android
    Dibuat menggunakan Java (atau Kotlin, opsional) di Android Studio. Tersinkronisasi penuh dengan database web. Tampilan native dengan navigasi yang smooth dan ringan. Fitur login/register, chat, dan notifikasi real-time.
    Teknologi Yang Digunakan
    HTML, CSS, JavaScript (JS) untuk tampilan dan interaktivitas web. Java untuk pengembangan Android. PHP & MySQL untuk backend server. C++ digunakan di sisi Android untuk kebutuhan performa (opsional/untuk modul tertentu seperti enkripsi atau AI ringan). API untuk komunikasi antara Android dan Web.
    Tujuan Proyek
    Mendemonstrasikan kemampuan dalam: Membangun sistem terintegrasi antara aplikasi Android dan Website. Menerapkan konsep full-stack development. Mengelola komunikasi data antara frontend dan backend dengan aman dan efisien.
